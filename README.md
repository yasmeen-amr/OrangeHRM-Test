
OrangeHRM Test Automation Project (Selenium)

This is a test automation framework built using Java, Selenium WebDriver, TestNG, and Maven to automate the testing of the OrangeHRM web application.

The framework follows the Page Object Model (POM) design pattern to ensure code reusability, maintainability, and scalability.

🚀 Key Features
Automated UI testing using Selenium WebDriver
Page Object Model (POM) design pattern
TestNG for test execution and assertions
Maven for project dependency management
Log4j for logging test execution details
Cucumber BDD (if applicable in your project)
Allure Reports for test reporting
Cross-browser testing support
📂 Project Structure
Pages → Page classes (POM design)
Tests → Test cases using TestNG
Steps → Step definitions (if using Cucumber)
runner → Test runner class
resources → Configuration files and feature files
allure-results → Test execution reports
🎯 Covered Scenarios
Login functionality testing
Admin module testing
My Info module testing
Recruitment module testing
🛠️ Tech Stack
Java
Selenium WebDriver
TestNG
Maven
Cucumber (if used)
Allure Reports
Log4j
📌 Objective

To ensure the quality and stability of the OrangeHRM application by automating functional test cases and reducing manual testing effort.
